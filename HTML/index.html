Live Face Off

Full Web Application

Live Face Off allows friends and family to connect with one another via webcam and enjoy some games together. This application was developed as a Beta testable prototype for Jonathan and Morgan. The mission behind Live Face Off is to give just about anyone the ability to easily login and rekindle old bonds or make new ones. Our focus is to take this application to the next level by continuously creating new content for users to try out, and also give more incentives by implementing achievements and an online matchmaking system.

This project was completed with a team of 4 developers. My main contribution was the Front End development of the 2 game components and the use of TokBox api component for live WebRTC video communication.

The main technologies I utilized were React.js, React Redux, React Router, Axios, WebSockets and Materialize CSS.
